# 💡 Rumo Game

> **모든 불을 끄는 클래식 로직 퍼즐**
> 깔끔한 UI와 애니메이션으로 즐기는 라이트 아웃 게임입니다.

### ✨ 주요 특징
* **간단한 규칙**: 터치한 타일과 인접한 타일의 상태가 반전됩니다.
* **깔끔한 디자인**: Lottie 애니메이션을 적용한 모던한 UI.
* **최적화**: Expo SDK 54 기반 안드로이드 완벽 대응.

### 🛠 기술 스택
* **React Native / Expo** (Router, SDK 54)
* **Lottie** (애니메이션)
* **EAS** (안드로이드 배포)

### 🚀 실행 방법
```bash
npm install
npx expo start
